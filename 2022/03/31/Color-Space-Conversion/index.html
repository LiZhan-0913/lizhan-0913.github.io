0. sRGB and Lab (Including XYZ)

本部分内容来源于ISO-17957-2015，该标准原用于评价LSC模块的校正效果。

(a) Convert sRGB values to linear RGB values in the range of 0 to 1

以R通道为例：

R’ = R_s / 255
if (R’ <= 0.04045)
R_lin = R’ / 12.92
else
R_lin = pow((R’ + 0.055) / 1.055, 2.4)

(b) Convert linear RGB values to XYZ

X 0.4124 0.3576 0.1805 R_Lin
Y = [0.2126 0.7152 0.0722] [ G_Lin ]
Z 0.0193 0.1192 0.9505 B_Lin
